## 符号说明

### 常用数学符号

- `∀` - 全称量词（对所有）
- `∃` - 存在量词（存在）
- `→` - 蕴含
- `⊆` - 子集
- `∈` - 属于
- `∪` - 并集
- `∩` - 交集
- `≤` - 偏序关系

### 系统符号

- `S` - 系统状态空间
- `T` - 追踪数据集
- `E` - 事件集合
- `→` - happens-before关系
- `G = (V, E)` - 调用图
