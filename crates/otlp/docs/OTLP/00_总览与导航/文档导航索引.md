## 🚀 快速开始

### 1. 安装依赖

```bash
# 确保使用 Rust 1.90+
rustup update
rustup default 1.90.0
```

### 2. 克隆项目

```bash
git clone https://github.com/your-repo/OTLP_rust.git
cd OTLP_rust
```

### 3. 运行示例

```bash
cd otlp
cargo run --example simple_demo
```

### 4. 查看文档

```bash
cargo doc --open
```
