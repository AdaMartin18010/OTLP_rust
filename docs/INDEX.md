## 📦 归档说明

本项目专注于 **4个核心主题**：

1. **`otlp`** - OpenTelemetry Protocol 实现
2. **`reliability`** - 统一可靠性框架
3. **`model`** - 理论模型实现库
4. **`libraries`** - Rust 生态成熟库集成

其他非核心主题（如 blockchain、edge_computing、ai_ml）已归档到 `ARCHIVE/` 目录。详见 [归档说明](../ARCHIVE/README.md)。
