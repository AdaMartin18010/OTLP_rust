## 🎯 核心概念速览

### 语义模型

- **操作语义**: 状态转换函数 `σ' = δ(σ, e)`
- **指称语义**: 语法到语义域的映射
- **公理语义**: 霍尔三元组 `{P} C {Q}`

### 执行流

- **Petri网**: 并发系统建模
- **依赖图**: 组件依赖关系
- **Actor模型**: 消息驱动并发

### 控制流

- **决策树**: 条件分支决策
- **策略模式**: 可插拔策略
- **PID控制**: 闭环反馈控制

### 数据流

- **流式管道**: 多阶段数据处理
- **时间窗口**: 滑动窗口聚合
- **数据血缘**: 追溯数据来源
