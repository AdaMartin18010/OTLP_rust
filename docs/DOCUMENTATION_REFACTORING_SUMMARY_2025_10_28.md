### 5. 代码质量

**标准要求**:

```rust
/// 函数文档注释
///
/// # 参数
/// * `param` - 参数说明
///
/// # 返回值
/// 返回值说明
///
/// # 示例
/// \`\`\`
/// let result = function(42);
/// assert_eq!(result, 84);
/// \`\`\`
///
/// # 注意
/// - 注意事项1
/// - 注意事项2
///
/// # 性能
/// - 时间复杂度：O(n)
/// - 空间复杂度：O(1)
fn function(param: i32) -> i32 {
    param * 2
}
```

**代码示例必须**:

- ✅ 可编译通过
- ✅ 包含完整注释
- ✅ 有预期输出
- ✅ 解释工作原理
- ✅ 说明性能特征
