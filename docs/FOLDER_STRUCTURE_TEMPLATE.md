# 文件夹标准结构模板

**版本**: 1.0  
**日期**: 2025年10月28日  
**用途**: 所有docs子文件夹的标准结构

---

## 📋 每个文件夹必须包含

### 1. README.md（必需）

**标准结构**:
```markdown
# [文件夹名称]

**版本**: X.Y
**创建日期**: YYYY-MM-DD
**更新日期**: YYYY-MM-DD
**状态**: ✅ 完整

---

## 📋 概述

[2-3段说明本文件夹的目的、内容和使用方法]

---

## 📂 文件结构

\`\`\`
folder_name/
├── README.md (本文件)
├── KNOWLEDGE_GRAPH.md (知识图谱)
├── COMPARISON_MATRIX.md (对比矩阵)
├── CONCEPTS.md (概念定义)
├── document1.md
├── document2.md
└── ...
\`\`\`

---
```

## 🎯 文档编号规则

### 文件命名

```
# 推荐格式
01_topic_name.md
02_topic_name.md
...

# 或主题分类
basic_01_introduction.md
basic_02_concepts.md
advanced_01_patterns.md
```

### 章节编号

```markdown
## 1. 第一章
### 1.1 第一节
#### 1.1.1 第一小节

## 2. 第二章
### 2.1 第一节
```
